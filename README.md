# Docker PHP Sample

Adapted from [Docker's PHP Language Guide](https://docs.docker.com/language/php/).

## Commands

Build and run container:
```
./scripts/watch.sh
```

Run composer update and save lock:
```
./scripts/update.sh
```

Run tests:
```
./scripts/test.sh
```

Remove container and volumns:
```
./scripts/down.sh
```
